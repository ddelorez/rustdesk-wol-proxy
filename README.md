# RustDesk WOL Proxy

Flask-based Wake-on-LAN API to wake RustDesk clients remotely.

## Status
- Initial repo setup
- Next: add app.py (Flask application)
- Then: systemd service for production

## Development
- Clone this repo on workstation
- Create venv: `python3 -m venv venv`
- Install deps: `source venv/bin/activate && pip install -r requirements.txt`
